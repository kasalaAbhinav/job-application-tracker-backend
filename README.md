# Job Application Tracker Backend

A backend application to track job applications, interview status, deadlines, company details, and notes.

## Tech Stack

- Java
- Spring Boot
- PostgreSQL / MySQL
- REST API
- JWT Authentication
- Docker

## Features Planned

- User registration and login
- Add job applications
- Update application status
- Filter applications by company/status
- Add interview notes
- Track deadlines
- REST API documentation

## Application Status Types

- APPLIED
- SHORTLISTED
- INTERVIEW
- OFFER
- REJECTED

## Database Entities Planned

### User
- id
- name
- email
- password

### JobApplication
- id
- companyName
- role
- status
- appliedDate
- deadline
- jobLink
- notes
- userId

## Goal

This project is built to practice Java Spring Boot, REST APIs, SQL, authentication, and backend deployment.
