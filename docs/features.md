# Features

## Phase 1: Basic Job Tracking

- Add a new job application
- View all job applications
- View one job application by ID
- Update job application status
- Delete job application

## Phase 2: Filtering and Search

- Filter applications by status
- Search by company name
- Search by role
- Sort by applied date

## Phase 3: User Authentication

- User registration
- User login
- JWT authentication
- Each user can only see their own applications

## Phase 4: Interview Tracking

- Add interview round details
- Store interview date
- Store interview notes
- Track offer/rejection status

## Phase 5: Deployment

- Add Docker support
- Add PostgreSQL database
- Deploy backend
- Add API documentation
